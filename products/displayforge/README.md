# DisplayForge

Visual presentation / signage / dashboards / rendered displays / kiosk-style output layer.

## Overview

DisplayForge is the visual presentation layer of the Forge ecosystem, handling:
- Digital signage and kiosk displays
- Dashboard rendering
- Real-time display updates
- Multi-screen coordination

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

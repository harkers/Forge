# Skill: Dashboard Layouts

## Scope
- Dashboard grid patterns
- Panel layout templates
- Widget composition
- Responsive scaling

## Capability

Design and implement dashboard layouts for operational displays.

## Key Requirements
- 2x2, 3x2, and custom grid layouts
- Panel resize and reflow
- Consistent widget spacing
- Visual hierarchy

## Layout Patterns

### Grid Layout
```
┌─────────┬─────────┬─────────┐
│ Panel 1 │ Panel 2 │ Panel 3 │
├─────────┼─────────┼─────────┤
│ Panel 4 │ Panel 5 │ Panel 6 │
└─────────┴─────────┴─────────┘
```

### Split Layout
```
┌──────────────────┬──────────┐
│                  │          │
│    Main Panel    │ Sidebar  │
│                  │          │
└──────────────────┴──────────┘
```

### Hero + Grid
```
┌─────────────────────────────┐
│         Hero Panel          │
├─────────┬─────────┬─────────┤
│ Small 1 │ Small 2 │ Small 3 │
└─────────┴─────────┴─────────┘
```

## Output
- Layout components
- Panel templates
- Grid system

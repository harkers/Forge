# RankForge

RankForge is a DevForge product focused on autonomous SEO, structured metadata, internal linking, image generation, and search-performance optimisation for WordPress.

## Purpose

RankForge is designed to evolve from a plugin into a closed-loop search optimisation system that can:

- optimise titles, meta descriptions, and schema
- generate and backfill featured images
- create Open Graph and Twitter image assets
- process large WordPress sites safely using queues
- add internal links based on semantic relevance
- use search performance data to improve future optimisation decisions

## Current Scope

The current implementation direction includes:

- Ollama-based SEO generation
- Stable Diffusion WebUI image generation
- dedicated SEO and image queues
- WordPress CLI support
- Open Graph and Twitter image metadata
- SEO-safe filenames and attachment metadata

## Planned Phases

1. Core Stability
2. Image Automation
3. Queue Architecture
4. SEO Intelligence
5. Internal Linking
6. Feedback Loop
7. Optimisation & Scale

## Repository Position

RankForge currently lives under the Forge umbrella repository as a DevForge-family product. It can be split into its own dedicated repository later if release cadence, code volume, or team workflow justify it.

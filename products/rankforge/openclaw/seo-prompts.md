# SEO Prompts

## Generate SEO Metadata

```
PROJECT: RankForge

TASK: Generate SEO metadata for a WordPress post.

INPUT:
- post title
- post content

REQUIREMENTS:
- generate meta title (max 60 chars)
- generate meta description (max 155 chars)
- generate focus keyword
- generate keyword variations
- ensure output is SEO-optimised and human readable

OUTPUT:
- JSON with meta_title, meta_description, keywords
```
